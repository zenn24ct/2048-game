# 🧩 2048風パズルゲーム

## 🎮 概要
- ブラウザ上で遊べる **シンプルな2048風パズルゲーム**  
- 同じ数字をスライドで合体させ、2048を目指す  
- 矢印キー / WASD / スワイプ操作に対応  
- PC・スマホの両方でプレイ可能  
- Undo（1手戻す）・Hint（簡易AIのおすすめ）機能付き  

---

## ⚙️ 操作方法
- **← / → / ↑ / ↓** ：タイルを動かす  
- **W / A / S / D** ：同じく方向操作  
- **スワイプ操作（スマホ）** ：対応済み  
- **New Game** ：新しくスタート  
- **Undo** ：1手前に戻る  
- **Hint (AI)** ：おすすめ方向を表示  

---

## 💻 技術構成
- **HTML / CSS / JavaScript** のみ（外部ライブラリなし）  
- **localStorage** でベストスコアを保存  
- **グリッド回転処理**（rotate関数）で4方向を統一的に実装  
- **requestAnimationFrame** なしで軽量ループ設計（都度描画）  
- **アニメーション（spawn / pop）** による出現・合体エフェクト

## 🪄 ゲームの仕組み（ざっくり）
- 4×4のマスに数字タイルを配置  
- 同じ数字がスライド方向でぶつかると合体（値×2）  
- 移動できなくなったら **Game Over**  
- 移動ごとにランダムで新しい「2」または「4」が出現  
- スコアは合体した値の合計

## 🧭 改修メモ（内部仕様）
- `moveLeft()` 関数で1方向を処理 → `rotate()` 関数で全方向を対応  
- タイル追加は `addRandomTile()` で空きセルからランダム抽出  
- 描画は `render()` と `placeTilesWithAnimation()` の2段階構成  
- AIヒントは「直後に最大スコアを得られる方向」を簡易計算
